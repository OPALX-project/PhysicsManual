34// ------------------------------------------------------------------------
36// An attribute parser defines the data type for an attribute. It is
37// used to parse the attribute, to read the attribute from the DOOM data
38// base, and to write it to that data base. It contains the name and
39// help text for the attribute. Optionally it may also contain a default
40// value for the attribute.
41// [p]
42// When ``is_readonly'' is true, the attribute cannot be redefined by
43// the user.
44// [p]
45// When ``is_deferred'' is true, the attribute must be re-evaluated
46// whenever it is used. This is the case for random error values.
47// When ``is_deferred'' is false, any expression for the attribute is
48// cached. It is re-evaluated only when any other definition has changed.
49
