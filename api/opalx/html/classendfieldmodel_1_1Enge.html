Detailed Description

Enge class is a symmetric Enge function for analytical field models

Enge function is

\(f(x) = 1/(1+exp(h(x-x0)))+1/(1+exp(h(-x-x0)))-1\).

where h is a polynomial in x/lambda with polynomial coefficients a
